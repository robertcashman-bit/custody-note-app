Custody Note web demo

The desktop app is the only supported way to record real client information. The web demo is published so prospective users can try the interface; it is not suitable for handling real custody notes, real client information, or any legally privileged or special-category data.

Why the web demo is not safe for real client data

  • The whole local database lives unencrypted in the browser's IndexedDB; anyone with access to your browser profile can read it.
  • Other websites that run JavaScript in this browser may, via supply-chain compromise of dependencies the browser loads, exfiltrate the database.
  • The page loads a third-party WebAssembly module from sql.js.org; the desktop app does not.
  • There is no encryption at rest, no recovery password, no idle lock, no audit log, and no encrypted backup.

Use the desktop app instead

Download the signed Windows installer from www.custodynote.com. The desktop app stores all data with AES-256-GCM at rest, supports a recovery password, can back up to UK-resident immutable storage, and meets the design requirements for legal-sector confidentiality.
